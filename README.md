# Veloce - Automotive Marketplace

Veloce is a full-stack car marketplace web application. Users can browse cars, filter listings, view car details, save cars and searches, compare vehicles, sell a car, read a buying guide, and use a small Gemini chatbot.

## Tech Stack

- Frontend: React, Vite, CSS
- Backend: Node.js, Express.js, REST API
- Database: SQL Server
- Authentication: JWT
- AI: Gemini API

## Main Features

- Car inventory and detail pages
- Filters by price, mileage, city, make, fuel type, and condition
- AI-style search
- Login and register
- Saved cars and saved searches
- Sell car page
- Compare cars page
- Buying guide
- Gemini chatbot

## Run Locally

Backend:

```bash
cd backend
npm install
npm run dev
```

Frontend:

```bash
cd frontend
npm install
npm run dev
```

Create `backend/.env` from `backend/.env.example`.

Do not upload `.env` to GitHub.
